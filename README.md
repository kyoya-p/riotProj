Firebase Cloud Functions
====

Setup
----
```
curl -sL https://firebase.tools | bash
firebase login
```
* ブラウザを勝手に開くのでログインする。ブラウザを開けない環境ならciモードで。
  https://firebase.google.com/docs/cli?hl=ja

* firebase-tools upgrade:  
`sudo npm install -g firebase-tools`
  
*  npm も upgradeが必要なら...  
`sudo apt upgrade`
